# Frontend Web - Sistema de Gestión Condominial

Aplicación web desarrollada con React, Vite, TypeScript y shadcn/ui.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en http://localhost:5173

## Build

```bash
npm run build
```

## Estructura

```
src/
├── components/    # Componentes reutilizables
│   └── ui/       # Componentes shadcn/ui
├── pages/        # Páginas de la aplicación
├── services/     # Servicios API
├── store/        # Estado global (Zustand)
└── lib/          # Utilidades
