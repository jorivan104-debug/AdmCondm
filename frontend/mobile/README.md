# Frontend Mobile - Sistema de Gestión Condominial

Aplicación móvil desarrollada con React Native y Expo.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start
```

Luego presiona:
- `a` para abrir en Android
- `i` para abrir en iOS
- `w` para abrir en web

## Estructura

```
src/
├── screens/     # Pantallas de la aplicación
├── components/  # Componentes reutilizables
├── services/    # Servicios API
└── store/       # Estado global (Zustand)
```

## Configuración

Asegúrate de actualizar la URL de la API en `src/services/api.ts` con la dirección correcta del backend.
